一.配置操作
1.设置用户信息
git config --global user.name  "name"
git config --global user.email "email"
2.0x80以上字符进行quote
git config --global core.quoepath false
二.基础使用操作
1.创建管理仓库
git init
2.查看仓库状态
git status
3.比较暂存区与工作区的差异
git diff filename
2.工作区与暂存区
 1> git commit "提交的注释"       | 1> git add filename
 将缓存区内容提交至工作区         | 将filename文件添加至暂存区
